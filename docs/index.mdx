# tsubu-cloud

**依存パッケージが勝手に外部と通信できない、WASM Component ホスティング**

tsubu-cloud では、`tsubu.json` に宣言した通信先だけが、アプリのビルド成果物
(WASM Component)そのものの構造として呼び出し可能になります。宣言していない
外部通信は実行時フィルタで弾かれるのではなく、コンポーネントの import として
そもそも存在しません。

- なぜ tsubu-cloud を使うのかは [Why tsubu-cloud](/why-tsubu-cloud) へ
- どこから始めればいいかは [Getting Started](/getting-started) へ
- 仕組み全体を理解したい場合は [Core Concepts](/core-concepts) へ
